jehan Mobile Wallet is a mobile banking services provided by jehan Bank. It is a very simple and convenient to use mobile banking system. Holder of any mobile device can use the service without internet connection but availability of a mobile network is required. It is cost free or the user is not charged by telecom operator for using the services.
